**Mi Portfolio – Antonio**

Este proyecto es un **portfolio web personal** creado para mostrar trabajos de ilustración y diseño gráfico con una presentación visual y cuidada. Es un proyecto académico desarrollado como parte de mis estudios, pensado además como una base escalable que podrá evolucionar conforme aumente mi nivel de programación.

** Enlace de Release **

https://github.com/antonioburon03/My-Portfolio-Antonio/releases/tag/v1.0.0
---

** Objetivo del proyecto**

- **Presentación**: Mostrar una selección de trabajos de forma clara y visual.
- **Identidad**: Construir una identidad personal a nivel gráfico y digital.
- **Aprendizaje**: Aplicar y profundizar en conocimientos de desarrollo web básico.
- **Escalabilidad**: Crear una base reutilizable para un portfolio profesional futuro.

---

** Tecnologías utilizadas**

El proyecto está desarrollado con tecnologías web fundamentales:

- **HTML5**: estructura semántica del contenido.
- **CSS3**: estilos y diseño, organizado en ficheros modulares.
- **JavaScript (vanilla)**: comportamiento, navegación por hash y modal de proyectos.

No se han usado frameworks externos: la idea es reforzar los fundamentos.

---

** Estructura del proyecto**

- **Archivo principal**: [index.html](index.html) – entrada y estructura del portfolio.
- **Estilos**: [assets/css](assets/css) – ficheros CSS por componentes (`index.css`, `layout.css`, etc.).
- **JavaScript**: [assets/js/main.js](assets/js/main.js) – inicializadores, router por hash y modal de proyectos.
- **Documentación**: [docs/project-brief.md](docs/project-brief.md), [docs/plan.md](docs/plan.md), [docs/project.yaml](docs/project.yaml).
- **Favicon y manifest**: [assets/favicon](assets/favicon) – iconos y `site.webmanifest`.
